overview

Techverse - IT Business React.js Template

Techverse is a React.js Template which is a high-Quality and well organized React.js Template specially designed to fit all the needs of IT Business and solutions. It will also be suitable for any kind of professional business template. It can be used for various business sites like startup, IT farm based service etc.

In this template has comments on each section included in the template so you can easily adapt your templates to suit your needs. This react.js template can be easily customizable. You can easily replace any picture with your own in just a few clicks.

It’s built with the most modern tools Tailwind CSS. it’s a very complete and highly customizable template that will help you build your own website layouts. It’s Simple, elegant, extensive, professional.

requirements

There are system requirements in order to install and setup the "Medico" and its components properly. Make sure that you are running the node.js install npm install. You can download Node.js from here: Node JS. NPM comes bundled with Node.js

Recommended node and npm

node install

npm install

Install Npm Packages

NPM : You must have to npm install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.

npm install

Development server

Run npm run start for a dev server. Navigate to http://localhost:3000/ . The app will automatically reload if you change any of the source files

npm run start

Build

Run npm run build to build the project. The build artifacts will be stored in the build/ directory. Use the npm run build --prod flag for a production build.

npm run build

npm run build --prod

More Help

To get more help on the react use npm --help or go check out the React Support.

npm --help

React installation

Please follow the instructions -

  1. For local host -

    1. Open your command prompt

    2. npm install

    3. npm run start

Change Basename and Route

You can change the route in App.js file. Go to /App.js file and make your route as your need.

Change Site Title and Favicon

Go to /public/index.html file to change the title and fav icon

Change Logo

Go to /src/components/navigation/Header.js file and change logo.

Replace /src/assets/images/logo.svg file to change logo.

Customize Menu

Go to /src/components/navigation/Header.js file and change menu items.

Customize Footer

Go to /src/components/navigation/footer/FooterTop.js file and change footer logo, social media, footer data.

Change Copyright Infomartaion

Go to /src/components/navigation/footer/FooterBottom.js file and change copyright information.

Section Components

Go to /src/components folder. In components folder you will get all the section components. Go through each components and change section data.

service components

Go to /src/components/service folder. Here you will find all the service related components.

Blog Components

Go to /src/components/blog and /src/components/blogDetails folder to find all the blog related components.

Social Networks

Go to /src/components/navigation/footer/FooterTop.js file to change social links.

Typography - Font Family

Go to /src/index.css and /tailwind.config.js file and put your font family link to change the font of the website.

Typography - Color

Go to /tailwind.config.js file and put your color code to change font color and other colors.
